INTELLIGENT TIMETABLE SCHEDULING SYSTEM
========================================

Description:
------------
AI-based system for automatic university timetable generation using Genetic Algorithms.

Requirements:
-------------
- Python 3.6 or higher (NO additional libraries needed)
- CSV file with timetable data

How to Run:
-----------
1. Place your data in 'timetable_data.csv'
2. Double-click 'app.py' or run: python app.py
3. Generated timetable will be saved as 'timetable_YYYYMMDD_HHMM.csv'

Data Format (CSV):
------------------
Class,Subject,Hours,Faculty,Code,Type
4A,Mathematics,4,Dr. Smith,MATH101,Theory
4A,Physics,3,Prof. Johnson,PHY101,Theory
4B,Chemistry,4,Dr. Williams,CHEM101,Lab

Features:
---------
- Genetic Algorithm optimization
- Conflict-free scheduling
- Balanced faculty workload
- Optimal room utilization
- No external dependencies
- CSV import/export
